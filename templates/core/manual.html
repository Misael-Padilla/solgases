Dashboard

El dashboard es la pantalla principal del sistema. Muestra cuatro métricas en tiempo real: productos activos, insumos activos, clientes registrados y ventas del mes actual. Debajo se muestran las últimas 5 ventas y las últimas 5 compras registradas. Si algún producto o insumo tiene stock por debajo del mínimo o por encima del máximo establecido (DA-002), el sistema muestra alertas de color naranja para que el administrador tome acción.

Productos

Catálogo de dotación y elementos de protección personal (EPP) que la empresa compra y vende. Cada producto tiene código único, categoría (Dotación / EPP), género, talla opcional, precio de compra y precio de venta.

Niveles de stock

Cada producto tiene un stock mínimo y un stock máximo. El sistema calcula automáticamente el nivel de inventario y lo muestra como indicador visual:

Nivel Condición Acción recomendada
BAJOStock actual ≤ stock mínimoReabastecer — aparece alerta en el Dashboard
NORMALStock entre mínimo y máximoInventario en rango óptimo
ALTOStock actual ≥ stock máximoSobrestock — aparece alerta en el Dashboard

Modificar stock

En operación normal, el stock se actualiza automáticamente al registrar compras y ventas. Si se requiere una corrección excepcional (ajuste de inventario físico, pérdida, etc.), solo el ADMIN puede modificar el stock desde el detalle del producto usando el botón "Modificar stock". El sistema exige ingresar un motivo obligatorio que queda registrado en el historial de stock del producto.

Activar / Desactivar

Solo el ADMIN puede activar o desactivar un producto. Al hacerlo, el sistema muestra un cuadro de confirmación donde es obligatorio escribir una observación que explique el motivo. Los productos inactivos no aparecen disponibles para nuevas ventas, pero su historial se conserva.

Insumos

Catálogo de gases industriales y productos químicos que la empresa adquiere para sus operaciones. Cada insumo tiene código único, subcategoría (Gas / Químico), unidad de medida (Litros, Kilos, Unidades), precio de compra y un proveedor principal de referencia.

Niveles de stock

Mismo sistema de umbrales que los productos (BAJO / NORMAL / ALTO). Las alertas de stock bajo y sobrestock de insumos también se muestran en el Dashboard.

Proveedor principal

Cada insumo tiene asignado un proveedor principal. Este dato sirve de referencia para saber a quién contactar cuando el stock esté bajo. En el desplegable solo aparecen los proveedores con estado Activo.

Modificar stock

Igual que en productos: el stock se actualiza automáticamente con las compras. Para correcciones manuales, solo el ADMIN puede modificar el stock desde el detalle del insumo. El motivo es obligatorio y queda registrado en el historial de stock.

Activar / Desactivar

Solo el ADMIN puede cambiar el estado de un insumo. Se requiere observación obligatoria. Los insumos inactivos no están disponibles para nuevas compras, pero su historial se conserva.

Compras

Registro de facturas de compra a proveedores. Cada factura puede incluir múltiples ítems (productos e insumos). Al guardar, el stock de cada ítem se incrementa automáticamente. Si el código de un ítem no existe en el sistema, se crea automáticamente con valores por defecto.

Cálculo de valores

Los campos Subtotal, IVA y Total se calculan automáticamente mientras se agregan ítems. El porcentaje de IVA predeterminado es 19% pero puede modificarse. Los valores calculados no son editables directamente — el sistema los recalcula al cambiar cantidades, precios o el porcentaje de IVA.

Activar / Desactivar factura

Solo el ADMIN puede cambiar el estado de una factura de compra. Se requiere observación obligatoria.

Efecto sobre el inventario: Al desactivar una factura, el stock de todos los ítems se revierte automáticamente (se restan las cantidades recibidas). Al activar una factura inactiva, el stock se restaura (se vuelven a sumar). Este mecanismo garantiza la coherencia del inventario cuando se anula una compra registrada por error.

Ventas

Registro de facturas de venta a clientes. Cada factura puede incluir múltiples ítems (productos e insumos). Al guardar, el stock de cada ítem se descuenta automáticamente. El sistema valida que haya stock suficiente antes de registrar cada ítem — si el stock es insuficiente, la factura completa no se guarda y se muestra un mensaje de error.

Métodos de pago

El sistema acepta dos métodos de pago: Efectivo y Transferencia Bancaria. Se puede registrar el nombre de la persona que recibe el pago en el campo Recibido por.

Cálculo de valores

Mismo mecanismo que en compras. Los campos IVA y Total se calculan automáticamente. El porcentaje de IVA predeterminado es 19%.

Activar / Desactivar factura

Solo el ADMIN puede cambiar el estado de una factura de venta. Se requiere observación obligatoria.

Efecto sobre el inventario: Al desactivar una factura, el stock de todos los ítems se restaura (se devuelven las cantidades vendidas). Al activar una factura inactiva, el sistema verifica primero que haya stock suficiente — si no lo hay, muestra un mensaje de error y la factura permanece inactiva. Si hay stock disponible, el descuento se re-aplica automáticamente.

Backup

Solo accesible para el rol ADMIN. Permite generar copias de seguridad manuales de la base de datos completa y consultar el historial de backups generados automáticamente por el sistema.
