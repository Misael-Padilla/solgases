Manual de usuario — SOLGASES

Guía de uso del sistema de gestión empresarial.

Dashboard

El dashboard es la pantalla principal del sistema. Muestra cuatro métricas en tiempo real: productos activos, insumos activos, clientes registrados y ventas del mes actual. Debajo se muestran las últimas 5 ventas y las últimas 5 compras registradas. Si algún producto o insumo tiene stock por debajo del mínimo o por encima del máximo establecido (DA-002), el sistema muestra alertas de color naranja para que el administrador tome acción.

Productos e Insumos

El stock se actualiza automáticamente al registrar compras y ventas. No se modifica manualmente en operación normal. Cada producto e insumo tiene un stock mínimo y máximo que define el rango óptimo de inventario.

Compras

Al registrar una factura de compra, el stock de los productos e insumos incluidos en los detalles se incrementa automáticamente. Si el código de un ítem no existe en el sistema, se crea automáticamente.

Ventas

Al registrar una factura de venta, el stock se descuenta automáticamente. El sistema valida que haya stock suficiente antes de registrar la venta. Solo se aceptan pagos en efectivo o transferencia bancaria.

Backup

Solo accesible para el rol ADMIN. Permite generar copias de seguridad manuales de la base de datos completa y consultar el historial de backups generados automáticamente por el sistema.
